הפרויקט  "משחק חידות" מכיל משחק חידות מהנה ובדיקת זמן
המשחק בנוי מכמה תיקיות 
time מטפל בנשואי הזמן משחק
player מטפל ביצירת אוביקט השחקן
game מטפל בריצת המשחק (הצגת שאלות בדיקתם וכו)
riddles אחראי על החידות ויצרת רשימה שמכילה את כל החידות
utils פונקציות עזר (כמו קריאה מהמשתמש)
